# Tableau-Installation-Guide-
This project provides a step-by-step guide to installing Tableau Desktop. It walks users through the complete process—from creating a business email to successfully installing and launching Tableau.  This guide is especially useful for beginners who are new to Tableau and need a structured installation process.  Author  Jeevan MS
Steps Covered
1. Business Email Creation
Create a new professional email account
Enter personal details and verify the account
Set up a secure password
2. Access Tableau Website
Navigate to the Tableau official website
Open the free trial download page
3. Enter Required Details
Fill in user information
Use the created business email
Submit the form to proceed
4. Download Tableau
Click on Download Free Trial
The installer file will be downloaded automatically
5. Install Tableau Desktop
Open the downloaded file
Accept terms and conditions
Click install and grant permissions
6. Launch and Register
Open Tableau after installation
Start the trial
Register using the business email
7. Completion
Tableau is successfully installed and ready to use

Some Basic Requirements
A valid email address (preferably business email)
Internet connection
Windows system (64-bit recommended)
